Blindtest

Projet site web
html5, css3, JavaScript, illustrator
Mai 2021
Lycée Jacques Prévert

En s'appuyant sur le texte de Vannevar Bush, j'ai conçu un site web plongé dans le noir pour faire vire à l'utilisateur, l'expérience d'une personne déficiante visuelle.
